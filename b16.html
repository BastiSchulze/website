Course Outline

Object-oriented design; Introduction to C++; Classes: data, methods, constructors; Data hiding: public vs private data, accessor methods, encapsulation; Inheritance; Polymorphism; Templates; Standard Template Library; Design Patterns.

By the end of the course students should:

1. Understand concepts of and advantages of object-oriented design including:

o Data hiding (encapsulation)

o Inheritance and polymorphism

o Templates.

2. Understand how specific object oriented constructs are implemented using C++.

3. Be able to understand C++ programs.

4. Be able to write small C++ programs.
